## Summary
Implement a Go HTTP server using the Gin framework that responds to GET /ping requests with JSON {"status":"ok"}. The solution will include a main.go file for the server implementation and a Makefile for building and running the server. Correctness will be verified by sending a GET request to the /ping endpoint and checking if the response is as expected.
